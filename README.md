WoodDye
==========
no plugin does wood more good!  Stain your wood with dyes - bleach it with bone meal!


### Configuration

    debugMode: false
Enable extra debugging messages in server logs.

### Commands

	/impact reload
Reloads current configuration.


### Permissions

none
